# PRD — Финансовые цели с AI («Клёво»)

> SPARC Phase 3 · SPECIFICATION · Product Requirements Document  
> Дата: 2026-04-14  
> Статус: Draft → Ready for Implementation

---

## Executive Summary

«Финансовые цели» — premium-функция Клёво для подписчиков PLUS. Пользователь создаёт цель накопления (отпуск, подушка безопасности, гаджет), вручную отмечает прогресс, а AI анализирует его реальные траты и даёт конкретные рекомендации: «Урежь траты на маркетплейсы на ₽2 000/мес — цель достигнешь на 3 месяца раньше». Монетизация: AI-советы только для PLUS. FREE получают 1 цель без советов.

---

## Problem Statement

Российский Gen Z откладывает деньги хаотично: нет визуальной цели, нет системы. Существующие приложения (Тинькофф Цели, Сбер Копилка) требуют интеграции с банком. Клёво работает с данными CSV, которые уже импортированы — использует их для AI-анализа без дополнительной привязки счёта.
