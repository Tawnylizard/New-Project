# Specification: Bot Onboarding

## User Stories

### US-1: Пошаговое приветствие
```
As a новый пользователь,
I want to получить понятное приветствие с объяснением что делать,
So that я понял ценность приложения и знаю следующий шаг.

Acceptance Criteria:
Given пользователь отправил /start впервые
When бот отвечает
Then показывается приветствие с описанием Клёво
And кнопки "Сбер" и "Т-Банк" для выбора банка
And кнопка "Другой банк"
```

### US-2: Инструкция по Сбербанку
```
As a пользователь Сбербанка,
I want to получить пошаговую инструкцию как скачать выписку,
So that я смог загрузить файл в приложение без ошибок.

Acceptance Criteria:
Given пользователь нажал "Сбер"
When бот отвечает
Then показывается инструкция: Сбербанк Онлайн → История → Выписка → CSV
And финальная кнопка "Открыть Клёво" ведёт в TMA
```

### US-3: Инструкция по Т-Банку
```
As a пользователь Т-Банка,
I want to получить пошаговую инструкцию как скачать выписку,
So that я смог загрузить файл в приложение без ошибок.

Acceptance Criteria:
Given пользователь нажал "Т-Банк"
When бот отвечает
Then показывается инструкция: Тинькофф → Счёт → Выписка → Excel/CSV
And финальная кнопка "Открыть Клёво" ведёт в TMA
```

### US-4: Другой банк
```
As a пользователь другого банка,
I want to узнать какие банки поддерживаются,
So that я понял ограничения и могу попробовать позже.

Acceptance Criteria:
Given пользователь нажал "Другой банк"
When бот отвечает
Then сообщение с текущими ограничениями
And предложение попробовать с Сбер/Т-Банк выпиской
```

### US-5: Команда /how
```
As a существующий пользователь,
I want to вызвать инструкцию в любой момент,
So that я мог перечитать как скачать выписку.

Acceptance Criteria:
Given пользователь отправил /how
When бот отвечает
Then показывается выбор банка (как в шаге 2 онбординга)
```

### US-6: Команда /banks
```
As a пользователь,
I want to узнать список поддерживаемых банков,
So that я знал работает ли мой банк.

Acceptance Criteria:
Given пользователь отправил /banks
When бот отвечает
Then список: ✅ Сбербанк, ✅ Т-Банк, 🔜 Альфа-Банк, 🔜 ВТБ
```

## Non-Functional Requirements

- Ответ бота: < 500ms
- Все тексты на русском языке
- Emoji используются умеренно (не более 2 на сообщение)
- Кнопки — не более 2 в ряд
- Длина одного сообщения — не более 1000 символов (читаемость)
