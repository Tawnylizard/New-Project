# PRD: Bot Onboarding — Клёво

## Overview

**Feature:** Пошаговый онбординг через Telegram Bot  
**Scope:** `apps/bot`  
**Priority:** must_have  
**Phase:** MVP patch

## Problem Statement

Текущий `/start` показывает одно приветственное сообщение с кнопкой открыть TMA. Пользователь не понимает:
- Что делать дальше
- Как скачать выписку из банка
- Чем Клёво отличается от других приложений

Результат: высокий churn на первом шаге, пользователи не доходят до загрузки CSV.

## Solution

Заменить одноэкранный `/start` на пошаговый онбординг:

1. **Приветствие** — кто такой Клёво, что умеет
2. **Выбор банка** — Сбер или Т-Банк (или другой)
3. **Инструкция** — как скачать выписку (шаги + скриншоты-описания)
4. **Призыв к действию** — открыть TMA и загрузить файл

## Target Users

Новый пользователь, впервые открывший бота. Возраст 18–28, российский Gen Z.

## Success Metrics

| Метрика | До | Цель |
|---------|-----|------|
| Доходимость до TMA | ~40% | >65% |
| Загрузка первой выписки | ~20% | >40% |
| Время онбординга | — | <2 мин |

## Scope

### In Scope
- Многошаговый `/start` с inline keyboard
- Инструкции Сбербанк (мобильное приложение)
- Инструкции Т-Банк (мобильное приложение)
- Команда `/how` — вызов инструкции в любой момент
- Команда `/banks` — список поддерживаемых банков

### Out of Scope
- Push-уведомления
- Онбординг внутри TMA (отдельная фича)
- Поддержка других банков (Альфа, ВТБ и т.д.)
