## Персонажи

### Persona A — Саша, 22 года, студент
- Подрабатывает курьером, тратит всё до копейки
- Использует Telegram 4–5 часов в день
- Смотрит ВК Клипы про «лайфхаки»
- Имеет карту Т-Банка и BNPL «Долями»
- **Боль:** «Не знаю, куда уходят деньги»
- **Мотивация:** Хочет поехать в отпуск летом

### Persona B — Катя, 26 лет, junior-разработчик
- Зарплата ₽80К, снимает квартиру
- Финансово тревожна, несколько подписок о которых забыла
- Активна в финансовых Telegram-каналах
- **Боль:** «Подписки съедают незаметно»
- **Мотивация:** Накопить на первый взнос по ипотеке

### Persona C — Денис, 19 лет, школьник/фрилансер
- Нерегулярный доход, живёт с родителями
- Тратит на игры, фастфуд, подписки
- Любит мемы и юмор
- **Боль:** «Деньги кончаются за неделю»
- **Мотивация:** Прикольно посмотреть, что скажет AI

## Вариант 1: «Вирусный Roast» — ОСНОВНОЙ ✅

> **Логика:** Пользователь приходит через развлекательный контент, пробует продукт «ради смеха», обнаруживает реальную ценность и конвертируется в платного пользователя.

---

### Фаза 1: ОСВЕДОМЛЁННОСТЬ (Awareness)

**Канал:** ВКонтакте Клипы / Telegram-каналы  
**Точка контакта:** Видео «AI отчитал меня за кофе» или пост «Жёсткий режим Клёво»

```
[Пользователь видит видео/пост]
      ↓
[Смеётся / узнаёт себя]
      ↓
[Нажимает ссылку на @klyovobot]
```

**Эмоция:** 😂 Веселье → 🤔 Любопытство  
**Действие пользователя:** Клик по ссылке в описании видео  
**Барьер:** Нет — ссылка открывает Telegram напрямую

**Микро-тренды в этой фазе:**
- [ВКонтакте Клипы: охват 18–24 лет вырос на 47% в 2024](https://vk.company/ru/press/releases/11309/) — выбираем ВК как первичный видео-канал
- [Telegram Mini Apps органическое обнаружение](https://telegram.org/blog/mini-apps) — каталог TMA как дополнительный канал

### Фаза 2: ПЕРВЫЙ КОНТАКТ (First Touch)

**Канал:** Telegram Mini App  
**Точка контакта:** Открытие @klyovobot, экран приветствия

```
[Открыли бота]
      ↓
[Экран: «Узнай, что думает AI о твоих тратах»]
[Кнопка: «Попробовать демо Жёсткого режима»]
      ↓
[Демо без регистрации: 3 примера roast'а]
      ↓
[CTA: «Загрузи свои траты — получи персональный roast»]
```

**Эмоция:** 😮 Удивление → 😄 Вовлечённость  
**Ключевое решение:** Демо работает без регистрации — снижает трение  
**KPI:** Доля пользователей, дошедших до CTA: > 60%

**Микро-тренды в этой фазе:**
- [Zero-friction onboarding в Telegram TMA](https://core.telegram.org/bots/webapps) — авторизация через Telegram initData, нет паролей
- [Финансовая тревожность → демо без данных снижает барьер](https://nafi.ru/analytics/finansovaya-trevozhnos-rossiyskoy-molodezhi-2024/)

### Фаза 3: ОНБОРДИНГ (Onboarding)

**Канал:** Telegram Mini App  
**Точка контакта:** 4-шаговый онбординг

```
Шаг 1: Авторизация через Telegram (1 клик)
      ↓
Шаг 2: «Как ты хочешь использовать Клёво?»
       [ Разобраться с тратами ] [ Накопить на цель ] [ Просто посмотреть ]
      ↓
Шаг 3: Загрузка данных
       [ Загрузить CSV выписки ] [ Ввести траты вручную ] [ Подключить Т-Банк API ]
      ↓
Шаг 4: Первый анализ (30 сек)
       → Топ-5 категорий трат
       → «Жёсткий режим» — первый roast
       → Найденные подписки-паразиты
```

**Эмоция:** 🤩 Восторг / 😳 Шок от реальности  
**Ключевой момент:** Первый roast должен быть метким и смешным — это aha-moment  
**KPI:** Завершение онбординга: > 70%

**Адаптация под РФ-рынок:**
- [CSV-экспорт из Сбербанк Онлайн](https://www.sberbank.ru/ru/person/accounts/statement) — инструкция в онбординге
- [CSV-экспорт из Т-Банк](https://www.tbank.ru/help/internet-bank/statement/) — инструкция в онбординге
- [Open Banking через СБП API — ЦБ РФ](https://www.cbr.ru/fintech/open_api/) — roadmap Phase 2
- Категоризатор знает: WB, Ozon, Яндекс Маркет, ВкусВилл, Самокат, Яндекс Еда, Самокат, Кикшеринг — [Data Insight РФ мерчанты](https://datainsight.ru/ecommerce)


## Вариант 2: «Боль после зарплаты»

> **Логика:** Пользователь приходит в момент финансового стресса — увидел выписку, узнал о долгах, получил отказ в кредите.

### Ключевые отличия от Варианта 1

| Фаза | Вариант 1 (Roast) | Вариант 2 (Боль) |
|---|---|---|
| Триггер прихода | Вирусный контент | Финансовый стресс |
| Эмоция входа | Любопытство / веселье | Стресс / стыд |
| Первый экран | «Жёсткий режим» (демо) | «Разберись с тратами» (утилита) |
| Тон онбординга | Игривый | Эмпатичный |
| Aha-moment | Смешной точный roast | «Нашёл, куда уходили деньги» |
| Конверсия в платное | Средняя | Высокая (пользователь мотивирован) |
| Retention | ★★★★☆ | ★★★☆☆ (после снятия боли падает) |

### Путь пользователя — Вариант 2

```
[Получил зарплату → через неделю пусто]
        ↓
[Гуглит: «куда уходят деньги приложение»]
        ↓
[Находит Клёво через SEO / Дзен-статью]
        ↓
[Онбординг: «Давай разберёмся»]
[Фокус: анализ трат, категории, подписки]
        ↓
[Aha-moment: «Оказывается, ₽6,400 уходит на подписки»]
        ↓
[Активно использует → подписывается на Плюс]
        ↓
[После решения проблемы → retention падает]
```

**Почему не выбран основным:** Канал привлечения (SEO/Google) медленнее и дороже Telegram-вирусности. Высокий churn после «лечения» боли.

**Когда актуален:** Как secondary path внутри основного CJM — пользователи, которые выбрали «Разобраться с тратами» на шаге 2 онбординга.

**Источники:**
- [Финансовая тревожность россиян — НАФИ 2024](https://nafi.ru/analytics/finansovaya-trevozhnos-rossiyskoy-molodezhi-2024/)
- [Каждый второй молодой россиянин не откладывает — Сбер 2024](https://www.sberbank.ru/ru/press_center/all/article?newsID=42ee7cb5-bc8d-4ff8-b38d-e26d2ef6afcd)

## Вариант 3: «Финансовая цель»

> **Логика:** Пользователь приходит с позитивной мотивацией — хочет накопить на конкретную цель.

### Ключевые отличия от Варианта 1

| Фаза | Вариант 1 (Roast) | Вариант 3 (Цель) |
|---|---|---|
| Триггер прихода | Вирусный контент | Позитивная мотивация |
| Эмоция входа | Любопытство | Воодушевление |
| Первый экран | «Жёсткий режим» (демо) | «Поставь цель» (конструктор) |
| Aha-moment | Смешной точный roast | «За 4 месяца накоплю на iPhone» |
| Конверсия в платное | Средняя | Высокая (цели = платный функционал) |
| Retention | ★★★★☆ | ★★★★★ (цель держит до достижения) |
| CAC | Низкий | Средний |

### Путь пользователя — Вариант 3

```
[Хочет накопить на iPhone 16 (₽89,000)]
        ↓
[Находит Клёво через Telegram-канал «Лайфхаки для студентов»]
        ↓
[Онбординг: «На что копишь?» → вводит цель и сумму]
        ↓
[AI считает: «Если сократить кофе на 50% — накопишь за 3.5 мес»]
        ↓
[Пользователь регулярно возвращается следить за прогрессом]
        ↓
[Подписывается на Плюс ради автосбережений]
        ↓
[После достижения цели — ставит новую]
```

**Почему не выбран основным:** Меньше виральности — «достиг цели» не так смешно публично делиться, как «AI меня осудил». Требует более сложного onboarding.

**Когда актуален:** Secondary path — пользователи, выбравшие «Накопить на цель» на шаге 2 онбординга.

**Источники:**
- [Геймификация и цели в банкинге — Markswebb 2024](https://markswebb.ru/report/mobile-banking-rank-2024/)
- [Т-Банк «Цели» рост 140% — 2023](https://www.tbank.ru/about/press-releases/goals-2023/)

## Сравнение вариантов

| Параметр | Вирусный Roast ✅ | Боль после зарплаты | Финансовая цель |
|---|---|---|---|
| **CAC** | Низкий (вирус) | Средний (SEO) | Средний-высокий |
| **Время до Aha-moment** | < 2 мин | 5–10 мин | 3–5 мин |
| **Конверсия free→paid** | 5–8% | 8–12% | 10–15% |
| **Retention 30d** | 35% | 25% | 45% |
| **Виральность** | ★★★★★ | ★★☆☆☆ | ★★★☆☆ |
| **Уникальность** | ★★★★★ | ★★★☆☆ | ★★★☆☆ |
| **Соответствие позиционированию** | ★★★★★ | ★★★★☆ | ★★★★☆ |
| **LTV** | Средний | Средний | Высокий |
| **Масштабируемость** | ★★★★★ | ★★★☆☆ | ★★★☆☆ |

**Итоговый вывод:** Вариант 1 выбран как основной из-за максимальной виральности и уникальности. Варианты 2 и 3 встроены как secondary paths после onboarding (выбор режима).
