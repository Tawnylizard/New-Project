# Product Requirements Document — «Клёво»

> Версия 1.0 · MVP · Дата: 2026-04-09

---

## 1. Обзор продукта

### Что такое «Клёво»?

**Клёво** — AI-финансовый ассистент для российской молодёжи 18–28 лет. Работает как Telegram Mini App. Анализирует траты пользователя и генерирует язвительные, но честные комментарии («Жёсткий режим»). Находит забытые подписки и BNPL-долги. Мотивирует экономить через юмор, а не скуку.

### Почему сейчас?

- 83% платежей в РФ — безналичные → данные о тратах уже цифровые
- BNPL вырос на 300%+ → молодёжь накапливает скрытые долги
- 67% Gen Z тревожится о деньгах, но продолжает тратить → разрыв между знанием и действием
- Telegram Mini Apps — нулевое трение установки для 82% аудитории
- Западные аналоги (Cleo AI, Mint) недоступны или не адаптированы под РФ

### Позиционирование

| Категория | Значение |
|---|---|
| **Продукт** | AI-финансовый ассистент |
| **Аудитория** | Молодёжь 18–28, РФ, Telegram-пользователи |
| **Проблема** | Не знают, куда уходят деньги; BNPL-долги накапливаются |
| **Решение** | «Жёсткий режим» — честный roast + анализ трат |
| **Дифференциатор** | Юмор + характер бренда + Telegram-native |
| **Цена** | Freemium; ₽199/мес за Плюс |

## 2. Целевая аудитория

### Первичная ЦА

**Молодёжь 18–28 лет, РФ:**
- Использует Telegram ≥ 2 часов в день
- Имеет карту Т-Банка, Сбера или другого российского банка
- Использует BNPL хотя бы раз в месяц (или знает о нём)
- Доход ₽20K–100K/мес
- Финансово тревожен, но не хочет «скучных» финансовых приложений

### Вторичная ЦА

- Студенты 17–22 лет с нерегулярным доходом
- Молодые специалисты 25–32 лет, ищущие инструмент для бюджетирования

## 3. Ключевые функции MVP

### 3.1 «Жёсткий режим» (Roast Mode)
AI анализирует траты и генерирует персональный язвительный комментарий.
- **Уникальность:** Конкретные цифры из данных пользователя, не шаблоны
- **Тон:** Дерзкий, но не оскорбительный (как «старший друг»)
- **Лимит (free):** 3 roast'а в месяц
- **Лимит (Плюс):** Без ограничений

### 3.2 Анализ трат
- Топ-5 категорий расходов за месяц
- Сравнение с предыдущим периодом
- Пай-чарт визуализация (Recharts)
- Адаптированные категории для РФ: WB, Ozon, Яндекс Еда, Самокат и др.

### 3.3 Охотник на подписки-паразиты
- Автоматически определяет рекуррентные платежи
- Показывает «мёртвые» подписки (давно не использовались)
- Подсчитывает годовую стоимость каждой подписки

### 3.4 BNPL-трекер
- Агрегирует обязательства по Долями, Сплит, Подели
- Показывает timeline предстоящих платежей
- Предупреждает если BNPL > 30% дохода

### 3.5 Шеринг и виральность
- Кнопка «Поделиться roast» в каждом результате
- Реферальная программа: месяц Плюс за каждого платящего друга
- Уникальные share-карточки с брендингом

## 7. Ограничения и допущения

**Ограничения:**
- Без лицензии ЦБ РФ → только информационный сервис (нет реальных транзакций)
- Все ПД в Yandex Cloud ru-central1 (ФЗ-152)
- Без Visa/MC → только МИР, СБП, ЮMoney
- Поддержка только русского языка в MVP

**Допущения:**
- Пользователи готовы загружать CSV банковских выписок
- YandexGPT способен генерировать качественный «roast» на русском
- Telegram Mini Apps остаются доступны без ограничений
- ЮKassa поддерживает auto-renewal для подписок
